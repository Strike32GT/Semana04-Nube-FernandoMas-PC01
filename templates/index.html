Web dockerizada y worker local ligero

Consulta ONPE Web

La web en Docker ahora solo recibe el Excel, administra trabajos y entrega el archivo final. La consulta ONPE se hace fuera del contenedor con un worker local que usa tu navegador real.

Nuevo flujo

1. Sube el archivo. 2. Ejecuta python worker_host.py en tu maquina. 3. Cuando termine, vuelve a esta pagina y descarga el Excel actualizado.

Ningun archivo seleccionado Maximo recomendado: {{ max_upload_mb }} MB
Esta arquitectura baja mucho el peso de la imagen Docker porque el navegador ya no va dentro del contenedor.
